Cloud Balance

Why sign in?

Cloud Balance is a check register along with features to compute your your monthly cash flow. Cloud Balance uses a bucket method to group expenses to quickly see where your money is going.

Income

Salary, Bonus, In most cases you want more.

Static

Cost you have little control over. Fixed.

Discretionary

Unique lifestyle choices you have control over.

Future

Money you save for future events.

The application is built to run in the cloud in Google's app engine. The technology stack includes Java, JSON, Angular JS, Javascript.
